# Installation
## Clone the Repo
```bash
git clone https://github.com/quesodeconte/point_app.git
cd point_app
```

## Install Dependencies
```bash
bundle install
```

## Run Database Migrations
```bash
rails db:migrate
```

## Run Database Seeding
```bash
rails db:seed
```
# Start the App
## Run Rails Server
```bash
rails s
```

## Navigate to the Login Page
http://localhost:3000/admin
